⚡ Block Size Method (memorise this)
/24=256 /25=128 /26=64 /27=32 /28=16 /29=8 /30=4
Network = IP rounded DOWN to multiple of block size
Broadcast = next network − 1 · First host = network+1 · Last host = broadcast−1
Example: 192.168.1.200/26 → block=64 → net=192.168.1.192 → bcast=.255 → hosts .193-.254
